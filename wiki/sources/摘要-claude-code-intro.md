# 摘要 - Claude Code 介绍

## 核心摘要

Claude Code 是 Anthropic 出品的官方 CLI 助手，三层扩展机制（Skill / Hook / MCP）让用户能把 AI 能力深度嵌入本地工作流。CLAUDE.md 是项目级别的全局契约入口。

## 关键要点

- Skill = 提示词模板，user-invocable 后变 slash command
- Hook = 事件触发的本地 shell 脚本
- MCP = 外部工具 / 数据源接入
