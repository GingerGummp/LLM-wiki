# Karpathy LLM-Wiki Gist 摘录（demo 占位）

> 实际仓库中此处放完整剪藏；demo 中放摘录占位以演示 ingest 流程。

LLM 时代的个人知识库应该是 markdown wiki，分四类：
- sources（原始资料摘要）
- entities（人物 / 工具 / 公司）
- concepts（框架 / 方法论）
- syntheses（综述）

不变量：
1. 原始资料只读
2. 每页强制双向链接
3. 矛盾不静默覆盖
